# Pan Asia Bank — Reconciliation & Exhibit Platform (POC)

Frontend-only prototype of the signed-off Reconciliation & Exhibit Platform.

## Run

```bash
cd web
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
